Templates mit fertigen Layouts

Selfhtml bietet mit diesen Vorlagen Beispiele für eigene Webseiten, die Sie frei verändern, erweitern und ausgestalten können.

Es besteht aus einem HTML-Template, das …

  • aus semantischen und validem Code besteht
  • leicht erweiterbar ist
  • barrierefrei und so auch für Screenreader bedienungsfreundlich ist.

Dazu gibt es mehrere Layouts (Skins oder Themes), die …

  • durch CSS-Stylesheets zentral für alle Seiten gesteuert werden
  • dank responsivem Webdesign auf allen Geräten gut aussehen

Für die Schulung bei der Fakultät 73 wurde diese Seiten überarbeitet, so dass damit Übungsaufgaben durchgeführt werden können. Außerdem wurden alle Stylesheets entfernt, damit die Schüler ein neues Layout erstellen können.
